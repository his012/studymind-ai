DashScope-compatible OpenAI SDK endpoint · model: qwen-plus. Generates chat replies & quiz content.
dashscope-intl.aliyuncs.com
🗂️
Memory Store
memory.py · ChromaDB persistent vector DB. Semantic retrieval per user_id + time/importance-based forgetting.
./studymind_db
Flow: Browser sends chat/quiz requests to FastAPI → Agent Engine pulls relevant past memories from ChromaDB (semantic search, scoped to user_id) → builds a personalized system prompt → calls Qwen-Plus via Alibaba Cloud's DashScope API → response returned to user, while new memories (subjects, weak topics, scores, deadlines) are auto-classified and saved back to ChromaDB. A forgetting pass prunes low-importance memories older than 30 days on every chat call.
